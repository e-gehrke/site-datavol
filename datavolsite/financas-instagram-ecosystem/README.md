# Ecossistema de Educação Financeira no Instagram

## Visão Geral

Este repositório centraliza a estratégia de marca, arquitetura de conteúdo, identidade visual e planejamento editorial de um ecossistema de perfis no Instagram dedicado à educação financeira.

O ecossistema foi desenhado para ocupar uma lacuna específica: ensinar finanças com rigor e clareza, sem linguagem de guru, sem promessas de enriquecimento e sem conteúdo raso. O objetivo é construir autonomia de raciocínio financeiro, não dependência de dicas ou cursos.

## Premissas Estratégicas

1. O ecossistema nasceu para durar. Nenhuma decisão de nome, conteúdo ou design deve parecer que existe para empurrar um produto.
2. Clareza é o diferencial. Não profundidade pela profundidade, mas profundidade que gera entendimento real.
3. Educação financeira não é aconselhamento financeiro. Os limites regulatórios e reputacionais estão documentados em `estrategia/compliance.md`.
4. O tom é de professor universitário que sabe ser acessível, não de influencer tentando parecer inteligente.
5. Escalabilidade desde o início. Cada série, cada pilar, cada nome foi pensado para crescer além do Instagram.

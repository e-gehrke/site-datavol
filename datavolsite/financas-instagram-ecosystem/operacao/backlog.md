# Backlog de Conteúdo

## Como Usar Este Arquivo

O backlog é uma lista viva de temas em desenvolvimento. Cada item deve ter:
- Tema e título provisório
- Perfil (Conta Certa / Nota Técnica)
- Série associada (se houver)
- Urgência: Normal / Sazonal / Urgente
- Status: Ideia / Em pesquisa / Roteiro pronto / Em produção / Publicado

Revisar semanalmente. Mover para "Publicado" com a data.

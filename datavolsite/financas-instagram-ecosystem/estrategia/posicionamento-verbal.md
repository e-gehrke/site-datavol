# Posicionamento Verbal

## a. Frase de Essência da Marca

**Quando você entende como os números funcionam, você para de ter medo deles.**

Essa frase não é um slogan. É o princípio editorial que guia todas as decisões de conteúdo. Nenhum post existe para gerar engajamento pelo engajamento. Todo post existe para transferir entendimento real.

Variações para uso em diferentes contextos:

- **Para o Conta Certa:** "Finanças sem economês. Números que fazem sentido na sua vida."
- **Para o Nota Técnica:** "Finanças com rigor. A mecânica por trás dos modelos."
- **Para o Fundamento (guarda-chuva):** "Da clareza no extrato à estrutura do modelo. Fundamentos financeiros para qualquer nível."

## d. Manifesto Curto

*Este texto é para uso em apresentações da marca, na bio do perfil Fundamento (quando existir), e como referência interna de posicionamento. Não deve aparecer em posts regulares.*

---

Finanças não é um idioma para iniciados.

Não precisa de vocação, de herança familiar ou de um assessor que cobra comissão para traduzir o que o extrato diz. Precisa de clareza sobre como os números funcionam.

A maioria do que assusta nas finanças é falta de exposição, não falta de inteligência. Quando você entende que juros compostos crescem exponencialmente, você para de achar que a dívida no cartão de crédito é gerenciável com o mínimo. Quando você entende o que é duration, você para de tratar todos os títulos de renda fixa como iguais.

Não ensinamos o que fazer. Ensinamos como funciona.

A diferença é fundamental.

## e. Tom de Voz

### Conta Certa

**Registro:** Coloquial sem ser informal demais. O tom de um professor que sabe que seu aluno é inteligente, só não teve exposição ao assunto.

**Características:**

- Usa exemplos numéricos reais. Não "R$ X mil". Usa "R$ 1.200" ou "R$ 847,60".
- Faz a pergunta que o leitor faria antes de respondê-la. "Mas espera, então o que acontece com o saldo devedor?"
- Explica o "por quê" antes do "como".
- Não usa superlativos sem evidência. Não "o melhor investimento" nem "o maior erro".
- Admite complexidade sem usá-la como desculpa. "Isso tem variações, mas o princípio básico é este."

**Analogias preferidas:** domésticas e concretas. Supermercado, conta de luz, salário, carro financiado, aluguel.

**Velocidade do texto:** Frases curtas. Parágrafos de até 3 linhas. Bullets quando há lista.

### Nota Técnica

**Registro:** Técnico sem ser acadêmico. O tom de um analista sênior escrevendo para o time, não para o cliente.

**Características:**

- Usa nomenclatura técnica corretamente e sem desculpas. Se o leitor não sabe o que é "spread de crédito", ele vai aprender pelo contexto.
- Cita a fórmula quando ela esclarece. Esconde quando ela confunde.
- Referencia fontes quando relevante (CPC 48, IFRS 9, Black-Scholes, CFA Institute).
- Distingue o que é consenso do que é metodologia específica. "Uma abordagem comum é X, mas há escolas que preferem Y porque..."
- É direto. Não usa introduções longas para dizer o que vai dizer.

**Analogias preferidas:** estruturais e metodológicas. Portfólios, fluxos de caixa, estruturas de capital, grades de risco.

**Velocidade do texto:** pode ser mais denso. O público aguenta e espera densidade.

## f. Palavras Proibidas

### Proibidas em ambos os perfis

| Palavra/Expressão | Motivo |
|---|---|
| Liberdade financeira | Clichê que não significa nada específico |
| Fique rico | Promessa sem base e sem regulação |
| Renda passiva | Usado de forma imprecisa e associado a esquemas |
| Independência financeira | Clichê com mesma contaminação de "liberdade financeira" |
| Mude sua vida | Linguagem de guru motivacional |
| Você merece | Apelo emocional sem substância |
| Segredo que os bancos escondem | Sensacionalista e falso |
| Todo mundo deveria saber | Gatilho de FOMO sem conteúdo |
| Faça seu dinheiro trabalhar | Clichê de educação financeira rasa |
| Mindset | Psicologia pop fora de contexto financeiro |
| É simples | Simplifica demais o que tem nuances |
| Compre agora | Qualquer variação de CTA agressivo |

### Proibidas no Conta Certa (por risco de interpretação inadequada)

| Palavra/Expressão | Motivo |
|---|---|
| Invista em X | Evitar sugestão direta de produto |
| O melhor investimento | Sem contexto individual, não existe "melhor" |
| Você vai ganhar X% | Promessa de retorno |
| Criptomoedas (sem contexto) | Área de alto risco regulatório |

### Proibidas no Nota Técnica (por desvio de posicionamento)

| Palavra/Expressão | Motivo |
|---|---|
| Dica do dia | Superficial, incompatível com o formato |
| Funciona assim para qualquer empresa | Overgeneralization técnica |
| Na prática não precisa disso | Deprecia rigor técnico que é o diferencial |

## g. Palavras Preferenciais

### Preferidas no Conta Certa

| Palavra/Expressão | Por quê funciona |
|---|---|
| Funciona assim | Indica mecanismo, não opinião |
| Na prática | Ancora no real, não no teórico |
| O número que importa aqui é... | Foca a atenção sem dramatizar |
| Depende de... | Honesto sobre contexto e variáveis |
| A diferença entre X e Y é... | Comparativo claro |
| Vamos calcular | Convida participação, não impõe |
| O que o banco não explica no contrato | Específico e jornalístico |
| Passo a passo | Estrutura didática sem infantilizar |

### Preferidas no Nota Técnica

| Palavra/Expressão | Por quê funciona |
|---|---|
| A hipótese aqui é... | Reconhece premissas explicitamente |
| O modelo assume que... | Honesto sobre limitações |
| A intuição por trás é... | Conecta fórmula ao raciocínio |
| Na prática, isso implica... | Liga teoria a consequência |
| A literatura sugere | Âncora acadêmica sem pedantismo |
| Sensibilidade ao parâmetro X | Linguagem técnica precisa |
| Ceteris paribus | Aceitável neste público |
| É uma simplificação, mas... | Honesto sobre trade-offs didáticos |
